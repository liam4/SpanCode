SpanCode is an HTML / CSS3 tool for making better (pretend) code displayers. Since all the customization happens in a single CSS3 file, you can make it look exactly the way you want. It doesn't check for syntax, so whether your code is in BASIC, Python, Java, or ActionScript - or even your own custom language, if you want! - it won't wonder why the code is like that - you are the one who makes the syntax.

Using SpanCode

The first thing your going to want to do is download spancode. You can get the default css file here. Next, you need to import it in your .html file. To do that, put this code into the <head> of your file:

  1. <head>
  2.  ...
  3.  <!-- LOAD SPANCODE -->
  4.  <link rel="stylesheet" type="text/css" href="spancode.css">

After that, you need to create a div with a class of "code". Make sure it's not a <code>, but a <div>.
