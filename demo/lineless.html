No Line-count in Code

  1. # You probably know this is line 1.
  2. # This is line 2...
  3.  
  4.  
  5. # Okay, so this is line 5.

Turns out it's good for doing things other than having fun with comments! :o For example, it could come in handy for representing the something that doesn't happen in any exact spot in the program:

  1. <body>
  2.  ...
  3.  <p>Yeah, yeah, this isn't in a span.</p>
  4.  ...
  5. </body>

To add it in, just remember this: Inside of your div class="code" is an ol. That stands for Ordered List, so all you need to do to make it lineless is add the attribute style="list-style: none" to it.

  1. <div class="code"><ol style="list-style: none;">
