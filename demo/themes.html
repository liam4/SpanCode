To load a dark theme, just use the following code in your HTML document:

  1. <link rel="stylesheet" type="text/css" href="themes/THEME NAME">

All your code will now be styled with that theme!

A list of themes can be found in the themes folder.

To create your own theme, create a copy of themes/theme_blank_light.css (or dark.css if you prefer that), and change the styling in it. Keep in mind that you need to change your href in your HTML file to load it!

  1. <link rel="stylesheet" type="text/css" href="my_theme.css">
